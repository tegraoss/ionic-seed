Configuring the project with vagrant:
=====================================

Initializing the VM:
-------------------

```
cd vagrant
vagrant up
```

Await until the server is conofigured, it may take a while

Access the VM with SSH:
---------------------

```
cd vagrant
vagrant ssh
```

Run the application on the browser
----------------------------

Access with ssh and..

```
cd /project
ionic serve
```

Run the app in an android device
----------------

```
ionic run android
```
